# GITHUB ISSUE TRACKER

🔹Live Site Link: https://github-issue-tracker-with-api.netlify.app/

I have completed this project within 3 days. It was an exciting journey with API. I updated the card section dynamically using API endpoints. I used vanilla JavaScript to make this website interactive.

Technology Stack:
🔹HTML5
🔹Tailwind CSS
🔹JavaScript (Vanilla)

Key Features: 
🔹Using the demo credentials, users can log in to the main page.
🔹Clicking different tabs will change the card count, the button colour, and update the UI. 
🔹Clicking the Open or Closed button will move the card to the related tab and update the count.
🔹Clicking any card will open a modal with details of the related card. 
🔹User can search their issue using the search bar.
