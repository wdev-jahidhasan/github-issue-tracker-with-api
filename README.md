# 🐛 GitHub Issue Tracker (API Based)

🔗 **Live Site:** https://github-issue-tracker-with-api.netlify.app/

This is an API-driven GitHub Issue Tracker web application where users can manage and explore issues dynamically. The project was completed within 3 days and focuses on building interactive UI using Vanilla JavaScript and API integration.

---

## 🚀 Features

- 🔐 Login system with demo credentials  
- 📊 Dynamic issue cards loaded from API  
- 🔄 Tab switching updates:
  - Card count  
  - Button color  
  - UI state  
- 📂 Open & Closed issue management  
- 📝 Click on any issue card to view detailed modal  
- 🔍 Search functionality to find specific issues  
- ⚡ Fully interactive and dynamic UI  

---

## 🛠️ Technology Stack

- HTML5  
- Tailwind CSS  
- DaisyUI  
- Vanilla JavaScript  
- DOM Manipulation  
- API Integration  

---

## ⚙️ How It Works

1. Login using demo credentials  
2. Browse issues in different tabs  
3. Click:
   - ✅ Open / Closed → to move issues between tabs  
   - 📄 Card → to see issue details in modal  
4. Use search bar to quickly find issues
