1️⃣ What is the difference between var, let, and const?
      Answer: var is basically function-scoped. It is possible to redeclare and reassign var. But still there is a hoisting-related issue.
      Let is block-scoped. Redeclare is not possible, but reassign is possible.
      const is also block-scoped. But it is fixed. Redeclare and reassign is not possible here.

2️⃣ What is the spread operator (...)?
      Answer: Spread operator is a syntax which spreads iterable elements as different values. It is needed when we have to use elements or values separately or merge them.

3️⃣ What is the difference between map(), filter(), and forEach()?
      Answer: All of map(), filter(), and forEach() works on array, but they are different in purpose and return method. map(), filter() return new array but forEach() returns nothing.

4️⃣ What is an arrow function?
      Answer: Arrow function is the shorter syntax of writing function which is alternative to 'function' keyword. There is no need to return manually in single line. It returns automatically. But if we use multi line, we have to use {} and return manually.

5️⃣ What are template literals?
      Answer: Template literals is one kind of syntax which helps to write multi-line codes in innerHTML and also helps to add dynamic value in UI. We have to use `` (Backticks) for writing multi-line codes and ${} for adding dynamic value.
